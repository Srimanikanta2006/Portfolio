# 🚀 Srimanikanta's Portfolio

A modern, interactive developer portfolio built with **React**, **TypeScript**, **Vite**, **Tailwind CSS**, **Three.js**, and **GSAP**. Showcasing projects, skills, awards, education, and more with beautiful animations and 3D effects.

---  

[Live Demo]([https://your-portfolio-demo-link.com](https://psmkportfolio-nine.vercel.app/))  

---

## ✨ Features

- **Animated Hero Section** with typewriter effect and 3D model
- **About** with interactive 3D background and tech ticker
- **Projects Showcase** with live links and tags
- **Skills Bento Grid** with categories and current learning
- **Education Timeline** with animated DNA helix
- **Awards** with animated cards
- **Stats** with animated counters
- **Contact Form** with EmailJS integration
- **Custom Cursor, Preloader, Page Transitions, Starfield Background**
- **Responsive** and **accessible** design
- **Open to Work** badge and social links

---

## 🛠️ Tech Stack

- **Frontend:** React, TypeScript, Vite
- **Styling:** Tailwind CSS, Custom CSS
- **3D & Animation:** Three.js, @react-three/fiber, GSAP, Anime.js, Framer Motion
- **UI/UX:** Custom Cursor, Preloader, Page Transitions
- **Email:** EmailJS
- **Other:** React Router, Lucide Icons

---

## 📦 Installation

```bash
# Clone the repo
https://github.com/Srimanikanta2006/portfolio.git
cd portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```

## 🧩 Main Sections & Components

- **Navbar:** Sticky, animated, mobile-friendly navigation
- **Hero:** Animated intro, typewriter, 3D model
- **About:** Tech ticker, 3D torus knot
- **Projects:** Cards with live/demo links, tags, and highlights
- **Skills:** Bento grid, categories, current learning
- **Education:** Timeline, animated DNA helix
- **Awards:** Animated cards for achievements
- **Stats:** Animated counters (CGPA, Projects, Hackathons, Awards)
- **Contact:** EmailJS-powered form, animated quote
- **Footer:** Social links, philosophy quote, author info
- **Preloader, Custom Cursor, Background Effects, Page Transitions**
